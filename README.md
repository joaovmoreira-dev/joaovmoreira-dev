## Projeto atual

### [Task & Project Management API](https://github.com/joaovmoreira-dev/Task-Project-Management-API)

API REST corporativa construída do zero, simulando um ambiente real de gestão de projetos e tarefas.

| | |
|---|---|
| **Runtime** | Node.js + TypeScript |
| **Framework** | Express |
| **ORM** | Prisma |
| **Banco** | PostgreSQL |
| **Auth** | JWT + bcrypt |
| **Infra** | Docker + docker-compose |
| **Testes** | Jest + Supertest |
| **Deploy** | Railway |

**O que foi implementado ao longo de 9 sprints:**

- Autenticação com JWT, expiração e proteção contra enumeração de usuários
- RBAC completo — ADMIN, MANAGER e MEMBER com regras de ownership por recurso
- CRUD de Projects e Tasks com validação de permissão por papel
- AuditLog registrando todas as ações críticas com rastreabilidade completa
- Segurança: rate limit, Helmet, CORS por ambiente, erros neutros
- 74 testes de integração cobrindo Auth, Projects, Tasks, RBAC e edge cases
- Documentação interativa com Swagger
- Ambiente containerizado com Docker

**[API em produção](https://task-project-management-api-production.up.railway.app/health)** · **[Swagger](https://task-project-management-api-production.up.railway.app/api-docs)**

## Próximo projeto — IIoT & Monitoramento Industrial

Tenho interesse em desenvolver um sistema de coleta de dados de máquinas industriais diretamente de CLPs,
com processamento, armazenamento e visualização em tempo real.

A ideia envolve:

- Comunicação com CLPs via protocolos industriais (Modbus, OPC-UA)
- Pipeline de ingestão e processamento de dados de telemetria
- Dashboard de acompanhamento com indicadores de desempenho em tempo real
- Geração de relatórios automáticos
- Foco em segurança de redes industriais (OT Security)
