## 📌 Projeto em destaque

### [Task & Project Management API](https://github.com/Joao-Vitor-Moreira/Task-Project-Management-API)

API backend corporativa construída do zero ao longo de 9 sprints.

- 🔐 Autenticação JWT com RBAC e Ownership
- 🏗️ Arquitetura em camadas (Controller / Service / Repository)
- 🛡️ Segurança: Helmet, CORS, Rate limiting, Auditoria de ações
- 🧪 50+ testes de integração com Jest + Supertest
- 🐳 Docker + Deploy no Railway
- 📖 Documentação interativa com Swagger

**[🔗 API em produção](https://task-project-management-api-production.up.railway.app/health)** · **[📖 Swagger](https://task-project-management-api-production.up.railway.app/api-docs)**
